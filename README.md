# UAS-Java
Sistem Bioskop
Pada sistem bioskop terdapat dashboard admin dan user, user dapat melihat informasi film yang sedang diputar dan dapat memesan tiket serta melihat riwayat pemesanan. pada sisi admin, admin dapat menambahkan film yang sedang diputar beserta informasi terkait film tersebut.

Fitur User
-Lihat Daftar Film
-Beli Tiket
-Lihat Riwayat Pembelian
-Logout

Fitur Admin
-Tambah Film
-Hapus Film
-Tambah User Baru
-Hapus User
-Logout
